Detailed Description

An container widget commonly known as an accordion.

It contains a pack of groups. The groups can be individually opened or closed. If the sum of heights of groups is greater than the height of the accordion, a scrollbar will appear.

Callbacks:
Callbacks are handled by the individual groups.
See also
Fle_Accordion_Group
Features:
  • Single/multiple open mode can be changed with single_open()
  • Groups can be added with add_group()
  • Groups can be removed with remove_group()
  • Groups can be inserted with insert_group()
  • All groups can be closed with close_all()
