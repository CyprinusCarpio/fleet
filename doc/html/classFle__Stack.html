Detailed Description

A container widget that stacks children vertically or horizontally.

A stack widget is a container that arranges its children in a vertical or horizontal stack. It is typically used to group related widgets together. It's very similar to Fl_Flex, the main difference is the ability to set minimum, preferred and maximum sizes for each child. The preferred size is effectively the maximum size during child widget positioning, except for cases where the sum of preferred sizes of all widgets is less than the available space. User dragging on the edges updates the preferred sizes.
