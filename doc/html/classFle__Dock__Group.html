Detailed Description

Represents a dockable group widget within a Fle_Dock_Host.

Fle_Dock_Group is a container widget (derived from Fl_Group) that can be attached to the edges of a Fle_Dock_Host or detached into its own floating window (Fle_Detached_Window). It manages its own state (docked/detached, orientation, locked, hidden), appearance (decoration, label, gripper), and interaction (dragging, resizing, pinning). It typically contains a primary content widget ("band widget") and provides standard controls like close and pin buttons. The behavior and appearance can be customized using various state flags (Fle_Dock_State).
