Detailed Description

A container widget for displaying property widgets.

A property sheet is a container widget that displays property widgets in a grid layout. It is typically used to group widgets pertaining to a specific object or entity.

void Fle_Property_Sheet::add_property_widget ( Fl_Widget *  w,
const char *  name 
)

Adds a widget to the property sheet with the given name. The widget's height dictates the height of the row. Other coordinates are ignored. Do not add widgets with visible labels, use the name parameter instead.

Parameters
wThe widget to add
nameThe name of the property. If it's an empty string, the widget will occupy both the name and value columns
